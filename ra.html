Raciocínio Algorítmico

O objetivo da disciplina é desenvolver a lógica de programação, facilitando o raciocínio na construção e entendimento de algoritmos, mostrando que eles estão presentes no dia a dia e, além de serem usados no cotidiano, são essenciais para computação.

Em termos mais técnicos, um algoritmo é uma sequencia lógica, finita e definida de instruções que devem ser seguidas para resolver um problema ou executar uma tarefa. Embora não se perceba, algoritmos são utilizados de forma intuitiva e auto diariamente quando executamos tarefas comuns, passando totalmente despercebido, porém estando presente o tempo todo, como é o caso de trocar uma lâmpada.

O raciocínio algorítmico é fundamental na criação de um algoritmo, pois tem, por objeto de estudo, as leis gerais do pensamento, e as formas de aplicar essas leis corretamente na investigação da verdade.

A lógica de programação e a construção de algoritmos são conhecimentos fundamentais para programar. Construído o algoritmo, pode-se, então, codificar um programa de computação em qualquer linguagem.

Desafio de indexação da Bíblia

Este desafio foi proposto no intuito de praticar conceitos aprendidos na disciplina de Raciocínio Algorítmico, pesquisando conteúdos e técnicas novas para completar os conhecimentos adquiridos.

A ideia por trás do desáfio é indexar arquivos de texto, sendo, neste caso, a Bíblia, utilizando-se de conteúdos como laços de repetição, dicionários, estruturas homogêneas e potencialmente outras estruturas de dadados mais complexas, dependendo bastante da busca feita.

Instruções do desafio:

  1. Ao iniciar o programa, o software deverá pedir ao usuário o nome do arquivo a ser indexado.
  2. Durante a indexação, o arquivo deverá ser quebrado em linhas. Para encontrar o término de linha, assuma um ponto final (“.”) ou uma quebra de linha computacional (“\n”).
  3. O software deverá ignorar acentos e a diferença entre letras maiúsculas e minúsculas.
  4. O software deverá requisitar ao usuário uma palavra a ser buscada, até que o comando seja digitado (assuma que este comando será digitado com os comparadores de menor (<) e maior (>)).
  5. Com cada palavra informada ao software, todas as frases do documento indexado deverão ser apresentadas.
  6. Dica: a indexação do arquivo deverá ocorrer apenas uma vez, isto é, quando o software é inicializado. Isso significa que a cada palavra informada pelo usuário, deverá ocorrer apenas a listagem das frases, e o arquivo original não deverá ser percorrido.
